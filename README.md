# <div align="center">GRAPHQL-CRUD<div>

## Installation
Install with npm:
```
$ sudo npm i knex -g
$ sh deploy.sh
```

## environment for production
``` 
NODE_ENV = production
MYSQL_HOST = your_host
MYSQL_USER = your_user
MYSQL_PASSWORD = your_password
MYSQL_DATABASE = your_dbname
```
